# transcript-youtube-video
Transcript an array of youtube videos into a JSON file


# Before running
You need to add urls to the allvideos.js file like this:
```
const allVideos = [
    "firstYoutubeUrl",
    "secondYoutubeUrl",
];

```

# How to run
```node 
getTranscription.js
```

Then a new ./transcription.json file will appear.
